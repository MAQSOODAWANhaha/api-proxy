## 模拟数据说明

由于后端API当前存在技术问题，前端实现了智能模拟数据服务：

### 自动检测机制
```typescript
// 应用启动时自动检测后端可用性
const isApiAvailable = await checkApiAvailability()
useMockData = !isApiAvailable
```

### 模拟数据内容
- **用户数据**：管理员账户信息
- **API密钥**：OpenAI、Claude、Gemini模拟密钥
- **统计数据**：请求量、成功率、响应时间等
- **系统信息**：服务版本、运行状态等
- **请求日志**：50条模拟日志记录

### 切换真实数据
一旦后端API问题解决，前端将自动切换到真实数据，无需代码修改。
